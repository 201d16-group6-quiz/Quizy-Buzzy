## What is Quizy-Buzzy?

It's a quiz generating and taking website with an interactive pages to create your own quiz and share it with other users  

**The website has these pages :**

1. Home
2. Categories(1-6 pages)
3. Creating quiz form page
4. About the team
5. `Registration`
